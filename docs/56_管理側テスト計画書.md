# zcomic-next 管理側 テスト計画書

| 項目 | 内容 |
|---|---|
| 文書ID | ZCN-ADM-TP-001 |
| 版数 | 1.0 |
| 発行日 | 2026-05-20 |

---

## 1. テストの目的

zcomic-next 管理サイト (apps/api-admin, apps/web-admin) の機能・非機能を確認し、旧PHP/旧Vite 版と同等以上の動作を保証する。

---

## 2. テスト範囲

- ログイン認証 (tb_adm) と平文→bcrypt 自動移行
- Home の3つの非同期ジョブ (解凍/結合/ER解凍)
- Home の検索結果操作 (移動/タイトル作成/Delete DB/Delete DB&Dir)
- Unknown 検索 (一覧 + マンガ王国検索 + 変更)
- 比較標準 / 比較E (比較取得 + stock 入換/削除 + 一括登録)
- Page取得 (13dl.me クロール + フォルダ作成)
- ファイル名 NFC/NFD 耐性 (macOS+Docker)
- cross-device 移動の完全性 (件数検証)

---

## 3. テスト種別

| 種別 | 内容 |
|---|---|
| 単体テスト | `parseArchiveBaseName` / `parseInnerFolderToVolume` / `parsePath` 等の純関数を Node で実行 |
| 結合テスト | GraphQL ミューテーション+DB+FS の連動を確認 (テストDB + テスト用 FS) |
| シナリオテスト | 実際の業務フロー (DL→解凍→比較→登録) を画面操作で実行 |
| 性能テスト | 数十巻のタイトルで移動・解凍時間が許容内 |
| 回帰テスト | 本サイト (apps/api/web) の動作に影響していないこと |

---

## 4. テスト環境

- ホスト: 192.168.40.99 (NAS / Docker Compose)
- コンテナ: zcomic-api-admin (port 4001) / zcomic-web-admin (port 8081)
- DB: MariaDB 10.11 の comic スキーマ (テストはバックアップ復元したコピー)
- ブラウザ: Chrome 最新 / Safari 最新 / Firefox 最新
- ファイルシステム: `/Volumes/public` 配下 (Comic/Renamed, Download/*)

---

## 5. 合否基準

- 画面別機能一覧 ([54_管理側画面別機能一覧.md](./54_管理側画面別機能一覧.md)) の全機能項目が動作
- 旧PHP/旧Vite と同等以上の挙動 (URL 等価性、機能等価性)
- Critical なバグ (データ消失/権限漏れ) ゼロ

- 期間: 構築完了後 5営業日

---

## 7. リスクと緩和

| リスク | 緩和 |
|---|---|
| cross-device 移動でファイル消失 | `cp -a` + ファイル数検証 + 失敗時 src 保持 |
| NFC/NFD で readdir が空 | `pickAccessiblePath` で3バリアント試行 |
| k-manga サイト構造変化で検索失敗 | 正規表現フォールバック + 診断エントリ返却 |
| パスワード平文残存 | 初回ログイン時に bcrypt 自動移行 |
