## 読み順の推奨

### ビューワー側 (新規参加者向け)

1. **基本設計書 (01)** — システム概要・機能要件・非機能要件
2. **画面別機能一覧 (09)** — 各画面の細かい挙動
3. **ビューワー追補 (10)** — 初版以降の差分
4. **システム構成図 (03)** — 物理配置・通信経路
5. **ER図 (04)** — DB設計
6. **詳細設計書 (02)** — 実装レベル
7. **テスト計画/項目 (05/06)** — テスト実行

### 管理側 (新規参加者向け)

1. **51 管理側基本設計書** — システム概要・機能要件
2. **54 管理側画面別機能一覧** — 画面の細かい挙動
3. **53 管理側システム構成図** — 物理配置・mount構成・リクエストフロー
4. **52 管理側詳細設計書** — 実装レベル / 主要モジュール
5. **55 管理側API一覧** — GraphQL Operations 一覧
6. **56 管理側テスト計画書 / 57 管理側テスト項目書**

## Mermaid 図を表示するには

`*.md` ファイルの図 (システム構成・ER図) は Mermaid 記法で記述されています。

- **VS Code**: 拡張機能「Markdown Preview Mermaid Support」
- **GitHub / GitLab**: そのまま表示
- **Obsidian**: 標準対応
- **Mermaid Live Editor**: https://mermaid.live
- **コマンドライン**: `mmdc -i 03_システム構成図.md -o out.svg`

## Swagger UI / OpenAPI (ビューワー側のみ)

`docker compose up -d` 後、以下から閲覧:

| URL | 内容 |
|---|---|
| `http://192.168.40.99:8080/docs` | Swagger UI (Web経由) |
| `http://192.168.40.99:4000/docs` | Swagger UI (API直) |
| `http://192.168.40.99:4000/openapi.yaml` | OpenAPI 3.1 YAML |
| `http://192.168.40.99:4000/openapi.json` | OpenAPI 3.1 JSON |
| `08_API仕様_redoc.html` | オフライン Redoc |

> 管理側は GraphQL のみで OpenAPI を提供していません。詳細は **55_管理側API一覧.xlsx** を参照。
