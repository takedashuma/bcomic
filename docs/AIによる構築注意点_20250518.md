## 1. 新着順 SQL 仕様の確定

ユーザー指定の参照SQLに合わせて comicFolders resolver を更新。

```sql
SELECT bok_vch6, bok_vch7,
       MAX(bok_txt1), MAX(bok_mid) AS max_bok_mid,
       MAX(bok_vch8) AS max_bok_vch8, COUNT(bok_vch8) AS cnt_bok_vch8
FROM tb_bok
WHERE bok_dlday IS NULL
  AND bok_vch9 = 'comic'
GROUP BY bok_vch6, bok_vch7
ORDER BY max_bok_mid DESC;
```

- GROUP BY を **日本語キー (authorJa=vch6, titleJa=vch7)** に変更
- WHERE に `vch9 = 'comic'` 追加
- ORDER BY を **MAX(id) DESC**（bok_mid の最大値降順）に変更
- お気入トグルもこの基準に合わせて日本語キーで動作するよう調整
