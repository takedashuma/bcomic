# 修正記録 2026-05-19

## 比較標準ページ (V_CompareComic) を再構築

旧PHP `/admin_new/comparNo.php?mode=COMIC` 系の機能を web-admin の 比較標準 (`/compare-normal`) ページに再実装した。

### 追加・変更したファイル

#### apps/api-admin (サーバサイド)

- `src/jobs/compareUnregist.ts` (新規)
  - REGIST_DIR 配下を深さ3で再帰スキャンし、葉フォルダ `/initial/[Author;著者] Title;タイトル/volume` を一覧化
  - 各エントリで `tb_bok` から `titleJa contains` で同タイトル巻の在庫 (stockVolumes) を取得
  - alreadyInDb / existingBokMid を付与して返却

- `src/jobs/compareOps.ts` (新規)
  - `exchangeDir(newDir)` … 旧 `exchangeDir.php` の代替。REGIST_DIR/<newDir> → COMIC_ROOT/<newDir> に入換 (rm + copy + rm)
  - `deleteDBandBook(bookPath)` … 旧 `deleteDBandBook.php` の代替。`prisma.volume.deleteMany({where:{folderPath:{contains:bookPath}}})` 後にフォルダ削除
  - `renameRegistFolder(oldDir, newDir, inRegist)` … 旧 `_folderRenane.php` の代替。同一 root 内 rename

- `src/jobs/registUnregist.ts` (新規)
  - 旧 `dirRead.php?mode=COMIC` の代替（非同期ジョブ化）
  - REGIST_DIR 配下を深さ3で走査し `tb_bok` に MAX+1 採番で INSERT
  - 完了後 `UNREGIST/* → COMIC_ROOT/` をコピーし UNREGIST を空にする

- `src/schema/typeDefs.ts`
  - `CompareEntry`, `UnregistCompareResult` 型を追加
  - Query: `compareUnregist: UnregistCompareResult!` を追加
  - Mutation: `exchangeDir`, `deleteDBandBook`, `renameRegistFolder`, `startRegistUnregistAll` を追加

- `src/schema/resolvers.ts`
  - 上記関数を import し、Query/Mutation に配線

#### apps/web-admin (フロントサイド)

- `src/gql/operations.ts`
  - `COMPARE_UNREGIST`, `EXCHANGE_DIR`, `DELETE_DB_AND_BOOK`, `RENAME_REGIST_FOLDER`, `START_REGIST_UNREGIST_ALL` の GraphQL operation を追加

- `src/routes/compare-normal.tsx` (全面書き換え)
  - 「比較取得」ボタン: REGIST_DIR をスキャンし結果テーブルを表示
  - 行毎のアクション: 入換 / 削除 / 変更 / LinkPage (`https://comic.k-manga.jp/search/word/<titleJa>` を別タブで開く)
  - 在庫表示: stockCount / stockVolumes / 同巻警告
  - 「NormalComic登録」ボタン: `startRegistUnregistAll` 非同期ジョブを起動し JobProgress で進捗表示
  - アクション完了行は灰色 + 取消線で視覚的にフィードバック

#### docker-compose.yml

- `api-admin` の COMIC_ROOT マウントから `:ro` を外す（入換・削除で書き込みが必要なため）

### 機能対応表

| 旧PHP | 新エンドポイント |
| --- | --- |
| `/admin_new/comparNo.php?mode=COMIC` | GraphQL `query compareUnregist` |
| `/admin_new/exchangeDir.php?newDir=…` | GraphQL `mutation exchangeDir(newDir)` |
| `/admin_new/deleteDBandBook.php?bookPath=…` | GraphQL `mutation deleteDBandBook(bookPath)` |
| `/admin_new/_folderRenane.php` | GraphQL `mutation renameRegistFolder(oldDir,newDir,inRegist)` |
| `/admin_new/dirRead.php?mode=COMIC` | GraphQL `mutation startRegistUnregistAll` (非同期ジョブ + polling) |
| LinkPage `comic.k-manga.jp/...` | 行右端のリンクボタン (`/search/word/<titleJa>`) |
